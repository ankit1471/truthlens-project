About TruthLens

TruthLens is an AI-Powered Fake News Detection System developed by Ankit Pal (Roll: 42, TYCS, Div: A). In an era of rampant misinformation, this project serves as a tool to help users discern the authenticity of news articles and social media posts.

Project Objectives

The primary objective of TruthLens is to build a web-based application that automatically classifies a piece of text as either REAL or FAKE. This is achieved by leveraging the power of Natural Language Processing (NLP) and Machine Learning techniques.

Technologies Used

  • Frontend: HTML, CSS, JavaScript (via Bootstrap)
  • Backend: Python with the Flask framework
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Natural Language Processing (NLP): NLTK
  • Database: SQLite for user management
